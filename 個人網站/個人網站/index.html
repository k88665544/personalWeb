自由自在控制CSS元素 - CSS選取器簡介

在 HTML 中，我們有時候會對同一個標籤給予多個 Class 名稱，像是 div class="one two"/div 而在 CSS 裡面則可能同時選擇多個 Class，像是 .one.two{} /*兩個 class 中有空格*/ .one.two{} /*兩個 class 中沒有空格*/ .one, .two{} /*兩個 class 中出現逗號*/ 對於初學 CSS 的人來說，常常會搞不清楚這三者的差異，究竟這三者有什麼不同呢...
